# Prediction-Market-

A production-ready decentralized Prediction Market platform similar to Polymarket with a modern UI, scalable architecture, and enterprise-level code quality.

## Tech Stack
- **Frontend:** Next.js 15 (App Router), TypeScript, Tailwind CSS, Shadcn UI, Framer Motion, Zustand
- **Backend:** NestJS, PostgreSQL, Prisma ORM, Redis, WebSockets
- **Contracts:** Solidity, Hardhat, OpenZeppelin
- **Web3:** Wagmi, Viem, MetaMask, WalletConnect
- **AI features:** Market summaries, Sentiment analysis, Probability, Scam detection
- **Oracles:** Chainlink
